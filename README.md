# 🎬 Netflix Content Analyzer & Recommender

## About
A data science dashboard to analyze and recommend Netflix content.

## Setup
1. Clone the repo
   git clone https://github.com/YourUsername/netflix-analyzer

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run app.py

## Features
- Data cleaning & EDA
- Genre, country, year visualizations
- AI-powered content recommender

## Tech Stack
- Python, Streamlit, Pandas
- Matplotlib, Seaborn, Plotly
- Scikit-learn (TF-IDF + Cosine Similarity)
